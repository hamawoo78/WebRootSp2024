  1. Android Sensor Overview


    2. Most Android devices come equipped with sensors that detect motion, orientation, and environmental conditions. These sensors provide precise data, helpful for monitoring device movement or environmental changes. For instance, games can utilize gravity sensors for user gestures, while weather apps use temperature and humidity sensors to calculate dew point, and travel apps use geomagnetic sensors for compass bearings.
    • Motion sensors
      These sensors measure acceleration forces and rotational forces along three axes. This category includes accelerometers, gravity sensors, gyroscopes, and rotational vector sensors.
    • Environmental sensors
      These sensors measure various environmental parameters, such as ambient air temperature and pressure, illumination, and humidity. This category includes barometers, photometers, and thermometers.
    • Position sensors
      These sensors measure the physical position of a device. This category includes orientation sensors and magnetometers.

  2. 13 kinds of sensors in mobile phones


    3. The sensor in a phone refers to the components of the phone that can be felt by the chip, such as distance, light, temperature, brightness, and pressure. These sensors are getting smaller and smaller, their performance is increasing stronger, and their cost is getting lower and lower, just like all electronic components.
    • Accelerometer: Detects motion, orientation, and vibration.
    • Gyroscope: Measures angular velocity and helps with motion tracking.
    • Magnetometer: Determines the device's orientation relative to the Earth's magnetic field.
    • Proximity Sensor: Detects nearby objects to control screen brightness and prevent accidental touches during calls.
    • Ambient Light Sensor: Adjusts screen brightness based on surrounding light conditions.
    • Barometer: Measures atmospheric pressure for weather forecasting and altitude detection.
    • Thermometer: Monitors device temperature for safety and performance optimization.
    • Humidity Sensor: Measures humidity levels in the environment.
    • Heart Rate Sensor: Tracks the user's heart rate for health and fitness applications.
    • Fingerprint Sensor: Enables biometric authentication for device security.
    • GPS: Determines the device's location for navigation and location-based services.
    • Microphone: Records audio for calls, voice commands, and other applications.
    • Camera Sensor: Captures images and videos, also used for augmented reality and facial recognition.

  3. Interaction composite sensors


    4. Some sensors are mostly used to detect interactions with the user. We don't define how those sensors must be implemented, but they must be low power and it's the responsibility of the device manufacturer to verify their quality in terms of user experience.

    • Wake up gesture
      A wake up gesture sensor enables waking up the device based on a device specific motion. When this sensor triggers, the device behaves as if the power button was pressed, turning the screen on. This behavior (turning on the screen when this sensor triggers) might be deactivated by the user in the device settings.
    • Pick up gesture
      A pick-up gesture sensor triggers when the device is picked up regardless of wherever it was before (desk, pocket, bag).
    • Glance gesture
      A glance gesture sensor enables briefly turning the screen on to enable the user to glance content on screen based on a specific motion. When this sensor triggers, the device will turn the screen on momentarily to allow the user to glance notifications or other content while the device remains locked in a non-interactive state (dozing), then the screen will turn off again.

  4. Environment sensors


    5. The Android platform provides four sensors which monitor relative ambient humidity, illuminance, ambient pressure, and ambient temperature near an Android-powered device.
    Unlike most motion sensors and position sensors, which return a multi-dimensional array of sensor values for each SensorEvent, environment sensors return a single sensor value for each data event.

    Table 1. Environment sensors that are supported on the Android platform.
    Sensor Sensor event data Units of measure Data description
    TYPE_AMBIENT_TEMPERATURE event.values[0] °C Ambient air temperature.
    TYPE_LIGHT event.values[0] lx Illuminance.
    TYPE_PRESSURE event.values[0] hPa or mbar Ambient air pressure.
    TYPE_RELATIVE_HUMIDITY event.values[0] % Ambient relative humidity.
    TYPE_TEMPERATURE event.values[0] °C Device temperature.1
